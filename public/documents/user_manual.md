# Manual do Usuário - GridZero (Bem-vindo ao Grid)

Você acaba de se conectar ao GridZero, um simulador de hacking de alta performance. Este manual o ajudará a navegar pelo sistema e se tornar um hacker de elite.

## 1. O Terminal

Sua interface principal é o terminal. Todas as ações consomem **recursos de hardware** e **energia**.

*   **Energia:** Necessária para iniciar qualquer operação. Você tem 100 pontos. Ela regenera 1 ponto a cada 5 minutos.
*   **Hardware:**
    *   **CPU:** Define a velocidade de processamento.
    *   **RAM:** Define a capacidade de multi-tarefa.
    *   **SSD:** Sua "saúde". Se chegar a 0%, você perde acesso. Falhas em hacks danificam o SSD.

## 2. Primeiros Passos

### 1. Escanear a Rede
Use o comando de scan para encontrar alvos vulneráveis na rede. Cada nó tem uma dificuldade (Level) e uma recompensa.

### 2. Iniciar um Hack
Ao encontrar um nó, você pode iniciar um ataque.
*   **Atenção:** Hacks levam tempo! A duração depende da dificuldade do servidor alvo.
*   **Perigo:** Enquanto você hackeia, você fica vulnerável por 60 segundos. Outros hackers podem te rastrear!

## 3. Evolução e Melhorias

### Ganhando XP e Nível
Hacks bem-sucedidos concedem XP. Ao subir de nível, seu hardware é atualizado automaticamente (upgrade de CPU e RAM), permitindo que você ataque alvos mais protegidos.

### O Mercado Negro (`shop`)
Use seus créditos acumulados para comprar programas avançados:
*   **KeyLoggers:** Para ganhar mais XP.
*   **Stealth:** Para ficar menos tempo exposto após um ataque.
*   **Crackers:** Para ataques sem custo de energia.
*   **AntiVirus:** Fundamental para limpar seu sistema após ser infectado.

## 4. Dicas de Sobrevivência

1.  **Não gaste toda sua energia de uma vez:** Sempre guarde energia para usar um AntiVirus caso seja atacado.
2.  **Monitore seu SSD:** Se o seu SSD estiver baixo, pare de hackear e procure formas de repará-lo (via upgrades ou comandos específicos, se disponíveis).
3.  **Fique de olho no Log:** Mensagens do sistema no terminal indicarão se você está sendo alvo de scan de outros usuários ou NPCs.
